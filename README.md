# MLB-Projects
MLB data and projects:

1. Offensive Player Projections (incomplete):

   Using pitch-level data collected using MLB Stats API with ball parks metrics (wall height, fence distance, elevation, etc.) and player speed (acceleration score, top speed average) collected from baseball savant to project MLB Offensive player perfomance.

2. Modeling Likelihood of Pitching and Identify which Pitcher was Subsuitted in (not started):

   Using pitch-level data collected from the MLB Stats API to model the likelihood of a pitching change and identify which pitcher was substituted.
